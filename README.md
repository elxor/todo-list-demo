# Todo List on VueJS

Todo List demo application on VueJS.

Live demo: [todo-list-demo](https://vuetodolistdemo.netlify.app/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
